# Number Guess Game

Pythonで作った1〜100の数当てゲームです。  
ユーザーが数字を入力し、大きい/小さい/正解を返すシンプルなCLIアプリです。

## How to Run

## Features
- ランダムな数字を生成
- 入力した数に対して “大きい/小さい” とヒントを返す
- 何回目で当てたか表示

## What I Learned
- 基本的な入出力処理
- while ループと条件分岐
- Pythonの標準ライブラリ（random）の利用

## Future Improvements
