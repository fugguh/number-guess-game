# 学習記録トラッカー（Learning Tracker）

学習内容と学習時間を記録・保存・一覧表示できるシンプルな Python CLI アプリです。  
JSON ファイルにデータを保存するため、アプリを終了しても記録が保持されます。

---

## 概要

このアプリは以下の操作ができます。

- 学習内容（タスク名）と学習時間（分）を追加
- 記録を一覧表示
- JSON ファイルへの保存と読み込み

学習進捗の管理や日々の勉強ログとして利用できます。

## 機能詳細

### 記録追加
- タスク名（空白不可）
- 学習時間（分）
  - 1 以上の整数のみ
  - 数字以外が入力された場合は再入力を促す

### 記録一覧
- 保存されている記録を以下の形式で表示

2025-01-25 - アルゴリズム学習 - 45分

yaml
コードをコピーする

---

## データ保存について

- `records.json` に Python の `json` モジュールを使って保存
- 起動時にファイルが存在しなければ自動的に新規作成
- 手動編集にも対応
